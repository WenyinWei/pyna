토로이달 물리 (``pyna.toroidal``)
=================================

``pyna.toroidal`` 은 자기 구속 domain namespace입니다. geometry, equilibria,
coordinates, field-line tracing, coils, perturbations, control metrics,
torus-deformation theory를 포함합니다.
