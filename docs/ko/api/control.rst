FPT 위상 제어 (``pyna.control``)
=================================

.. note::
   ``pyna.control`` 은 **일반적인** FPT 위상 제어 framework를 담고 있으며,
   면적 보존 동역학계라면 어느 분야에도 적용할 수 있습니다. 토로이달 전용
   제어 helper(gap response, island control, q-profile)는
   :mod:`pyna.toroidal.control` 을 사용하세요.
