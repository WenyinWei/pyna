FPT-Topologiesteuerung (``pyna.control``)
=========================================

.. note::
   ``pyna.control`` enthält das **generische** Framework zur
   FPT-Topologiesteuerung, anwendbar auf jedes flächenerhaltende dynamische
   System.  Für toroidal-spezifische Steuerhilfen (Spaltantwort,
   Inselsteuerung, q-Profil) verwenden Sie :mod:`pyna.toroidal.control`.
