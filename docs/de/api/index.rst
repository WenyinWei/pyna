API-Referenz
============

.. note::

   ``pyna.toroidal`` ist der in pyna dokumentierte Namensraum für toroidale
   und magnetische Geometrie.  Historische Kompatibilitätsschichten
   ``pyna.MCF`` wurden entfernt.

Die handgeschriebenen Seiten unten erläutern die stabilen öffentlichen
Schnittstellen.  Die vollständige, aus dem Quellcode geparste Referenz wird
unter :doc:`/en/api/generated/pyna/index` erzeugt, wenn ``sphinx-autoapi`` installiert
ist.
