Toroidale Physik (``pyna.toroidal``)
====================================

``pyna.toroidal`` ist der Domänen-Namensraum für magnetische Einschließung.  Er
enthält Geometrie, Gleichgewichte, Koordinaten, Feldlinienverfolgung, Spulen,
Störungen, Steuerungsmetriken und Theorie der Torusdeformation.
