Toroidal Physics (``pyna.toroidal``)
====================================

``pyna.toroidal`` is the magnetic-confinement domain namespace.  It contains
geometry, equilibria, coordinates, field-line tracing, coils, perturbations,
control metrics and torus-deformation theory.
