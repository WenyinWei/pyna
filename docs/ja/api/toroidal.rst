トロイダル物理（``pyna.toroidal``）
==================================

``pyna.toroidal`` は磁場閉じ込め領域の名前空間です。幾何、平衡、座標、磁力線追跡、
コイル、摂動、制御指標、torus-deformation 理論を含みます。
