FPT トポロジー制御（``pyna.control``）
======================================

.. note::
   ``pyna.control`` には、任意の面積保存力学系に適用できる **汎用** FPT
   トポロジー制御フレームワークが含まれます。トロイダル専用の制御 helper
   （gap response、island control、q-profile）には :mod:`pyna.toroidal.control`
   を使ってください。
