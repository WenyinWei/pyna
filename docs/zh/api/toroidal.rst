环形磁约束命名空间 (``pyna.toroidal``)
======================================

``pyna.toroidal`` 是推荐的环形磁场、坐标变换、平衡和控制命名空间。它在
通用拓扑对象之上加入 ``R/Z/phi``、winding、monodromy、cyna 加速截面切割
和磁坐标工具。

常见入口：

- 解析或数值平衡；
- PEST、Boozer、Hamada、equal-arc 坐标；
- 岛链、流形、RMP 响应和控制；
- torus deformation 解析工具。
