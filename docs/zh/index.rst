pyna 中文文档
=============

**pyna** 是用于动力系统分析和磁约束聚变物理研究的 Python 工具包。它同时
覆盖环形磁场线追踪、Poincare 映射、Hamiltonian 系统、N-body 相互作用、
有限维映射、Ito 随机微分方程，以及把采样数据提升为相空间几何对象的统一
拓扑词汇。
