Учебные материалы и примеры
===========================

Публичные ноутбуки сгруппированы по рабочим процессам. Сборка документации
копирует ``notebooks/`` в дерево исходников Sphinx, поэтому пути ниже повторяют
структуру репозитория.

Рекомендуемая траектория обучения
---------------------------------

Начните с :doc:`/ru/quickstart`, затем пройдите рабочий процесс общей геометрии,
стохастические модели, а после этого тороидальные примеры монодромии/RMP:

1. :doc:`/ru/mini-cases`
2. :doc:`sde-monte-carlo`
3. :doc:`/notebooks/i18n/ru/tutorials/RMP_resonance_analysis`
4. :doc:`/notebooks/i18n/ru/tutorials/monodromy_xcycle_analytic`
5. :doc:`/notebooks/i18n/ru/tutorials/island_jacobian_analysis`
6. :doc:`/notebooks/i18n/ru/tutorials/RMP_island_validation_solovev`

Общие динамические системы
--------------------------

.. toctree::
   :maxdepth: 1

   sde-monte-carlo

Рабочий процесс общей геометрии и рабочий процесс неподвижных точек
аналитического стелларатора теперь включены в учебник по RMP-резонансу, а не
публикуются как отдельные текстовые ноутбуки. Этот учебник показывает ту же
цепочку геометрической интерпретации: выборочные пересечения -> геометрия
неподвижных точек -> классификация X/O -> наложения многообразий и координатной
сетки.

Стохастические дифференциальные уравнения
-----------------------------------------

Учебник SDE предварительно выполняется локально, потому что оценка распределений
часто использует десятки или сотни тысяч путей Монте-Карло. GitHub Pages
рендерит сохраненные выводы вместо траты CI-времени на тяжелые ячейки
выборки.

RMP, острова и анализ Пуанкаре
------------------------------

При изучении магнитной топологии начните с ноутбука анализа резонансов. Теперь
он покрывает бездивергентные RMP-шаблоны, важную ветвь ``m=1``, проверку
неподвижных точек ``cyna``, многокомпонентные атласы магнитного спектра
контравариантного радиального поля ``B^r``, модульные карты резонансов
``q``/``m/n`` с необязательными проекциями Пуанкаре и островными наложениями,
смешанные спектры RMP/nRMP, полный nRMP-ответ от всех нерезонансных мод,
модуляцию скорости силовых линий и проверки порядка возмущений.

Классические и общие динамические системы
-----------------------------------------

Репозиторий также содержит легкие ноутбуки в ``notebooks/examples``:
``Lorenz_attractor.ipynb``, ``resonance_1_1_map.ipynb``,
``Mobiusian_saddle_cycle.ipynb``, ``Xcycle_construction.ipynb`` и
``FPT_DX_to_DP_sympy.ipynb``. Они сохраняются как исходные примеры, а не как
выполненные страницы документации, потому что несколько из них остаются
черновыми ноутбуками без заголовков разделов.

Статические иллюстрации учебников
---------------------------------

Некоторые более длинные рабочие процессы представлены в репозитории как
статические рисунки и сгенерированные выводы в ``notebooks/tutorials``. Они покрывают
диагностику q-профиля, координаты PEST/Boozer/Hamada/Equal-arc, сканирования
подавления островов, управление фазой, многообразия Пуанкаре и примеры
равновесия Соловьёва с одной нулевой точкой.
