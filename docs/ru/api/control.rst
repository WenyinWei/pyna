FPT-контроль топологии (``pyna.control``)
=========================================

.. note::
   ``pyna.control`` содержит **универсальный** фреймворк FPT-контроля
   топологии, применимый к любой динамической системе, сохраняющей площадь. Для
   тороидально-специфичных вспомогательных средств управления (gap response,
   island control, q-profile) используйте :mod:`pyna.toroidal.control`.
