Тороидальная физика (``pyna.toroidal``)
=======================================

``pyna.toroidal`` - это пространство имен домена магнитного удержания. Оно
содержит геометрию, равновесия, координаты, трассировку силовых линий, катушки,
возмущения, метрики управления и теорию деформации тора.
