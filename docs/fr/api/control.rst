Controle de topologie FPT (``pyna.control``)
============================================

.. note::
   ``pyna.control`` contient le cadre **générique** de contrôle de topologie
   FPT, applicable a tout systeme dynamique preservant l'aire. Pour les
   auxiliaires de contrôle spécifiques au toroïdal (réponse de gap, contrôle
   d'ilot, profil q), utilisez :mod:`pyna.toroidal.control`.
