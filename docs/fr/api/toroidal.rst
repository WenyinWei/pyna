Physique toroidale (``pyna.toroidal``)
======================================

``pyna.toroidal`` est l'espace de noms du domaine du confinement magnétique. Il
contient la géométrie, les équilibres, les coordonnées, le traçage de lignes de
champ, les bobines, les perturbations, les métriques de contrôle et la théorie
de deformation du tore.
