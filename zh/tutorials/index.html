教程和示例#

公共 notebook 按工作流分组。文档构建会把 notebooks/ 复制到 Sphinx 源树， 因此下面的路径与仓库布局一致。

推荐学习路径#

先读 快速开始，再依次学习通用几何工作流、随机模型，以及环形 Monodromy/RMP 示例：

  1. 迷你案例

  2. SDE 蒙特卡洛分布

  3. RMP 仿星器共振分析

  4. 解析 X 型鞍点周期轨道的 Monodromy 矩阵与轨道位移

  5. 磁岛链 Jacobian 分析：周期轨道与 Monodromy 矩阵

  6. RMP 磁岛宽度验证：Solov’ev 解析平衡

一般动力系统#

  • SDE 蒙特卡洛分布

通用几何工作流和解析仿星器固定点工作流现在已经并入 RMP 共振教程， 而不是作为独立的纯文本 notebook 发布。该教程展示同一条提升链： 采样交点 -> 不动点几何 -> X/O 分类 -> 流形与坐标网格叠加。

随机微分方程#

SDE 教程会在本地预执行，因为分布估计通常使用数万或数十万条蒙特卡洛路径。 GitHub Pages 渲染保存的输出，而不是在 CI 上消耗时间运行重型采样 cell。

磁坐标和平衡#

  • 托卡马克平衡中的磁坐标系

RMP、磁岛和 Poincaré 分析#

研究磁拓扑时，请从共振分析 notebook 开始。它现在覆盖无散度 RMP 模板、 重要的 m=1 分支、cyna 固定点验证、多分量逆变 B^r 磁谱图册、带可选 Poincaré 和磁岛叠加的模块化 q/m/n 共振图、混合 RMP/nRMP 谱、来自所有 非共振模的总 nRMP 响应、磁力线速度调制，以及扰动阶数检查。

  • RMP 仿星器共振分析
  • RMP 磁岛宽度验证：Solov’ev 解析平衡
  • 磁岛链 Jacobian 分析：周期轨道与 Monodromy 矩阵

RMP_resonance_exec.ipynb 保留在仓库中，作为共振分析工作流的 执行/cache 变体，但公开文档链接到上面的解释版本。

  • Möbius 鞍点循环：倍周期与负特征值
  • 解析 X 型鞍点周期轨道的 Monodromy 矩阵与轨道位移

经典和一般动力系统#

仓库还在 notebooks/examples 下包含轻量 notebook： Lorenz_attractor.ipynb、resonance_1_1_map.ipynb、 Mobiusian_saddle_cycle.ipynb、Xcycle_construction.ipynb 和 FPT_DX_to_DP_sympy.ipynb。它们作为源示例保留，而不是作为已执行文档页面发布， 因为其中几个是没有章节标题的草稿式 notebook。

静态教程图#

若干较长工作流在仓库中以 notebooks/tutorials 下的静态图和生成输出表示。 它们覆盖 q 剖面诊断、PEST/Boozer/Hamada/等弧长坐标、磁岛抑制扫描、 相位控制、Poincaré 流形和 Solov’ev 单零点示例。
