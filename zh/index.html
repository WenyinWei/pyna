pyna - Python DYNAmics#

pyna 是一个面向 动力系统分析 和 磁约束聚变物理 的 Python 库。它覆盖场线追踪、Poincare 映射、Hamiltonian 系统、N-body 相互作用、 有限维映射、Ito 随机微分方程，以及把采样数据提升为相空间几何对象时共用的 拓扑词汇。

安装、验证 cyna，并运行第一个环形场线和通用动力系统示例。
