자기섬 Jacobian 분석: 주기 궤도와 Monodromy 행렬#

이 노트북은 해석적 스텔러레이터 평형에서 자기섬 사슬에 대한 전체 Jacobian/Monodromy 분석을 보여 줍니다. Julia MCF_scripts 접근법을 pyna 스타일로 옮긴 예입니다.

과학적 배경#

주기적 자기력선 궤도 또는 순환은 정확히 \(n\)번의 토로이달 회전 뒤 시작점으로 돌아오는 자기력선입니다: \(X(\phi+2\pi n)=X(\phi)\).

이는 Poincaré 사상 \(P^n\)의 고정점입니다. 공명 \(q=m/n\)에 대해, 여기서 \(q=B_\phi r/(B_{pol}R)\)이면 궤도 주기는 \(m\)회 회전입니다.

Jacobian 행렬 \(DX(\phi)\)는 다음처럼 진화합니다.

\[\frac{dDX}{d\phi}=A(r(\phi),z(\phi),\phi)\cdot DX, \quad DX(\phi_0)=I\]

여기서 \(A_{ij}=\partial(R B_{pol,i}/B_\phi)/\partial x_j\)입니다.

Monodromy 행렬 \(M=DP(\phi_0+2\pi n)\)은 선형화된 \(n\)회전 사상을 줍니다. \(M\)의 고유값은 다음처럼 해석됩니다.

  • \(|\lambda|=1\): elliptic, O점, stable

  • \(|\lambda|>1\): hyperbolic, X점, unstable

1. 설정: q=2/1 자기섬 사슬을 갖는 StellaratorSimple 구성#

m_h=2, n_h=1, epsilon_h=0.02를 사용해 q=2 공명을 만듭니다. 이 모델에서는 q=m_h/n_h=2이고 자기섬 사슬의 주기는 2, 즉 토로이달 2회전입니다.

q 프로파일: q(r)=q0+(q1-q0)*psi=q0+(q1-q0)*(r/r0)^2

  • r=0에서 q=q0=1.5

  • r=r0에서 q=q1=3.5

  • q=2 곡면: psi=(q-q0)/(q1-q0)=0.5/2.0=0.25, 따라서 r/r0=0.5

3. O점에 대한 Monodromy 분석#

Monodromy 행렬 \(M=DP(\phi_0+2\pi n)\)는 주기 궤도의 안정성을 인코딩합니다. O점, 즉 elliptic 점에서는 고유값이 단위원 위에 있습니다: \(|\lambda|=1\).

Greene residue: \(R_G=(2-\text{Tr}(M))/4\)

  • \(0<R_G<1\): elliptic, stable

  • \(R_G<0\): hyperbolic, unstable, 표준 경우

  • \(R_G>1\): 반사를 동반한 hyperbolic

# field_func_2d wrapper: stellarator.field_func([R,Z,phi]) returns unit tangent
# We need (R,Z,phi) -> (dR/dphi, dZ/dphi)
def field_func_2d(R, Z, phi):
    tang = stellarator.field_func(np.array([R, Z, phi]))  # [dR/ds, dZ/ds, dphi/ds]
    dphi_ds = tang[2]
    if abs(dphi_ds) < 1e-15:
        return np.array([0.0, 0.0])
    return np.array([tang[0] / dphi_ds, tang[1] / dphi_ds])

5. 섭동의 효과: \(\delta B\) 아래 궤도 이동#

작은 helical 섭동 \(\delta B\)를 진폭 epsilon_pert로 더합니다. 궤도 변위는 다음을 만족합니다.

\[\frac{dX_{cyc}}{d\phi}=A\cdot X_{cyc}+\delta b_{pol}\]

여기서 \(\delta b_{pol}=[R\delta B_R/B_\phi - R B_R \delta B_\phi/B_\phi^2; \text{Z에도 동일}]\)입니다.

[11]:

epsilon_pert = 0.005  # perturbation amplitude

# Build perturbed stellarator (slightly different helical phase to create delta B)
stellarator_pert = StellaratorSimple(
    R0=3.0, r0=0.35, B0=1.0,
    q0=1.5, q1=3.5,
    m_h=2, n_h=1, epsilon_h=epsilon_h + epsilon_pert,
)

def delta_field_func(rzphi):
    """Perturbation field: difference between perturbed and unperturbed."""
    f0 = np.asarray(field_func(rzphi))
    f1 = np.asarray(stellarator_pert.field_func(rzphi))
    # Note: field_func returns unit tangent vectors, not raw B fields.
    # For the orbit displacement, we use the difference in the phi-parameterized field.
    # The perturbation in terms of dR/dphi, dZ/dphi:
    g0 = np.array([f0[0]/f0[2], f0[1]/f0[2], 1.0])
    g1 = np.array([f1[0]/f1[2], f1[1]/f1[2], 1.0])
    return g1 - g0

6. 안정성 도표: Greene residue vs 섭동 진폭#

epsilon_pert를 0에서 0.02까지 스캔합니다. 각 진폭마다 다음을 수행합니다.

  • O 순환과 X 순환을 찾는다.

  • Greene residue \(R_G=(2-\text{Tr}(M))/4\)를 계산한다.

전이 \(R_G:0\to1\)은 자기섬 붕괴, 즉 KAM-like transition을 나타냅니다.

[13]:

print('Computing stability diagram (scanning epsilon_h)...')
epsilon_arr = np.linspace(0.005, 0.04, 12)

greene_O = []  # O-point Greene residues
greene_X = []  # X-point Greene residues
valid_eps = []

for eps in epsilon_arr:
    st_eps = StellaratorSimple(
        R0=3.0, r0=0.35, B0=1.0,
        q0=1.5, q1=3.5,
        m_h=2, n_h=1, epsilon_h=eps,
    )
    ff = st_eps.field_func

    # Find X-point near known seed (2.8763, -0.1237)
    orbit_x = find_cycle(
        ff, np.array([2.8763, -0.1237, 0.0]),
        n_turns=2, dt=0.15, RZlimit=RZlimit,
        max_iter=30, tol=1e-8, n_fallback_seeds=6, fallback_radius=0.02,
    )

    # Find O-point near known seed
    psi_r = st_eps.resonant_psi(2, 1)[0]
    r_r = np.sqrt(psi_r) * st_eps.r0
    opoint_seed = np.array([st_eps.R0 + r_r*np.cos(-2.32), r_r*np.sin(-2.32), 0.0])
    orbit_o = find_cycle(
        ff, opoint_seed,
        n_turns=2, dt=0.15, RZlimit=RZlimit,
        max_iter=30, tol=1e-8, n_fallback_seeds=8, fallback_radius=0.02,
    )

    rg_x = None
    rg_o = None

    if orbit_x is not None:
        M_x = orbit_x.DPm
        rg_x = (2.0 - np.trace(M_x)) / 4.0

    if orbit_o is not None:
        dist_axis = np.sqrt((orbit_o.rzphi0[0]-st_eps.R0)**2 + orbit_o.rzphi0[1]**2)
        if dist_axis > 0.05:
            M_o = orbit_o.DPm
            rg_o = (2.0 - np.trace(M_o)) / 4.0

    if rg_x is not None or rg_o is not None:
        valid_eps.append(eps)
        greene_X.append(rg_x)
        greene_O.append(rg_o)
        rg_x_str = f'{rg_x:.4f}' if rg_x is not None else 'N/A'
        rg_o_str = f'{rg_o:.4f}' if rg_o is not None else 'N/A'
        print(f'  eps={eps:.3f}: R_G(X)={rg_x_str}, R_G(O)={rg_o_str}')

print(f'Computed {len(valid_eps)} data points')

요약#

이 노트북은 다음을 보여 주었습니다.

  1. Poincaré 사상에 대한 Newton-Raphson을 이용한 주기 궤도 찾기

  2. 변분방정식을 통한 궤도 위 Jacobian 진화

  3. det(M) ~ 1, 즉 면적 보존을 갖는 Monodromy 행렬 \(M=J(\phi_0+2\pi n)\)

  4. 안정성 지표로서 Greene residue \(R_G\)

  5. 주기 경계조건을 이용한 섭동 아래 궤도 이동

  6. 섭동 진폭에 따라 자기섬 사슬 안정성이 어떻게 바뀌는지 보여 주는 안정성 도표

핵심 결과:

  • O점은 \(0<R_G<1\)을 갖는다(elliptic, stable).

  • X점은 \(R_G<0\) 또는 \(R_G>1\)을 갖는다(hyperbolic, unstable).

  • det(DPm(phi)) ~ 1이 전체 구간에서 유지되어 면적 보존, 즉 Hamiltonian 구조를 확인한다.

이 분석은 private stellarator의 실제 평형 데이터 없이도 같은 물리 알고리즘을 해석적 스텔러레이터에서 재현할 수 있음을 보여 줍니다.
