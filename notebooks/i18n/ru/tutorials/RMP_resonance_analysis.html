Анализ резонансов RMP в аналитическом стеллараторе#

Этот ноутбук - основной публичный рабочий процесс для геометрии резонансов в аналитическом стеллараторе. Он объединяет два прежних текстовых учебника в единый визуальный расчет:

  1. Построить аналитическое равновесие стелларатора и трассировать сечения Пуанкаре.

  2. Вычислить резонансные Фурье-компоненты RMP и их аналитические неподвижные точки X/O.

  3. Превратить сырые точки сечения в геометрическое описание: пересечения, маркеры неподвижных точек, резонансные поверхности, радиальные полосы ширины островов в O-точках, локальные устойчивые ветви и координатные наложения.

  4. Сравнить невозмущенные и возмущенные сечения с сеткой в стиле PEST.

  5. Свести ширины островов, перекрытие Чирикова и спектр \((m,n)\).

  6. Создать современную многосекционную фигуру с помощью утилит pyna.plot.

Ноутбук предназначен для локального выполнения перед публикацией документации. GitHub Pages отображает сохраненные выводы вместо повторного вычисления трассировок силовых линий.

[EQ] Построить равновесие стелларатора#

Используется аналитический стелларатор с одной винтовой гармоникой:

  • Большой радиус \(R_0 = 3.0\) м, малый радиус \(r_0 = 0.3\) м, поле на оси \(B_0 = 2.5\) Тл

  • Линейный профиль \(q\): \(q_0=1.5\) (ось) -> \(q_1=4.5\) (LCFS)

[RMP_NRMP_WORKFLOW] Разложение RMP/nRMP#

Мода возмущения на магнитной поверхности становится физически осмысленной только после сравнения с локальным вращательным преобразованием силовой линии. При выбранном соглашении Фурье

\[F(\theta,\varphi)=\sum_{m,n} F_{mn}\exp[i(m\theta+n\varphi)],\]

локальная расстройка равна

\[\Delta_{mn}=m\iota+n, \qquad \iota=1/q.\]

часть спектра

условие

геометрический эффект

рабочий процесс pyna

RMP

abs(m*iota+n) <= tol

открывает цепочки островов; X/O-точки, полосы ширины в O-точках и сепаратрисные ветви являются осмысленными объектами первого порядка

find_resonant_components_analytic, compare_cyna_fixed_points_for_component, draw_rmp_resonance_section

nRMP

abs(m*iota+n) > tol для каждой нерезонансной моды

все нерезонансные моды складываются в гладкую деформацию магнитной поверхности и модуляцию скорости силовой линии

fieldline_velocity_spectrum_on_circular_surface, FieldlineVelocitySpectrum.nonresonant_response()

смешанный спектр

присутствуют оба типа

резонансная островная геометрия накладывается на полную nRMP-деформированную поверхность

compose_magnetic_perturbations, nonresonant_response, rmp_nrmp_mode_rows для диагностики

валидация

векторное поле должно быть физическим

несоленоидальный шаблон может создать ложную топологию

radial_rmp_field_template, circular_shell_divergence_diagnostic

Ключевое различие: диагностика RMP фокусируется на резонансных строках спектра, а отклик nRMP является полной суммой по всем нерезонансным строкам:

\[\delta r(\theta,\varphi)=\sum_{m\iota+n\ne 0}\frac{F_{r,mn}}{i(m\iota+n)}\exp[i(m\theta+n\varphi)].\]

Таблицы вкладов полезны для ранжирования и проверки сходимости, но модель задается полным нерезонансным спектром, а не выбранной главной компонентой.

[RMP_FIELD] Задать и визуализировать поле возмущения RMP#

Применяется одномодовый RMP с базовой модой \((m,n)=(2,1)\) и амплитудой \(\delta B=1\) мТл. Пользовательский шаблон имеет вид

\[B_r = B_\mathrm{rmp}\cos(m\theta-n\varphi+\alpha),\]

но radial_rmp_field_template также добавляет компенсирующие полоидальные/тороидальные компоненты, нужные для бездивергентности полного цилиндрического векторного поля в локальной метрике круговой оболочки. Показанная радиальная проекция остается привычным резонансным драйвом \(\delta B^r\).

Та же утилита поддерживает важную ветвь m=1; в этом случае требуется тороидальная компонента, потому что радиальная дивергенция содержит часть, независимую от \(\theta\).

def field_func_perturbed(rzphi_1d):
    """Unit-tangent dRZphi/ds for the field-line ODE with RMP added."""
    rzphi_1d = np.asarray(rzphi_1d, dtype=float)
    R, Z, phi = rzphi_1d[0], rzphi_1d[1], rzphi_1d[2]
    theta = np.arctan2(Z, R - R0_eq)
    psi = eq.psi_ax(R, Z)
    q = float(eq.q_of_psi(psi))
    r_minor = np.sqrt((R - R0_eq)**2 + Z**2)
    B_phi = eq.B0 * eq.R0 / R
    B_pol = B_phi * r_minor / (R * max(abs(q), 1e-3))
    if r_minor > 1e-10:
        BR0 = -B_pol * np.sin(theta)
        BZ0 =  B_pol * np.cos(theta)
    else:
        BR0 = BZ0 = 0.0
    delta_BR_eq = eq.epsilon_h * eq.B0 * psi * np.cos(eq.m_h * theta - eq.n_h * phi)
    db = delta_B_RMP(R, Z, phi)
    BR_tot = BR0 + delta_BR_eq + db[0]
    BZ_tot = BZ0 + db[1]
    B_phi_tot = B_phi + db[2]
    B_mag = np.sqrt(BR_tot**2 + BZ_tot**2 + B_phi_tot**2) + 1e-30
    return np.array([BR_tot/B_mag, BZ_tot/B_mag, B_phi_tot/(R*B_mag)])

[NONRESONANT_DEFORMATION] Нерезонансная рябь как деформация магнитной поверхности#

Винтовая рябь в этом аналитическом равновесии не является резонансной компонентой RMP, открывающей остров (m,n)=(2,1). Тем не менее она изменяет геометрию соседней магнитной поверхности. Если сравнивать ньютоновские неподвижные точки с недеформированной круговой поверхностью, это гладкое смещение выглядит как искусственная ошибка фазы.

Здесь мы дискретизируем вклад винтовой ряби в ОДУ силовых линий,

\[\frac{dr}{d\varphi}=F_r(\theta,\varphi), \qquad \frac{d\theta}{d\varphi}=\iota+F_\theta(\theta,\varphi),\]

выполняем преобразование Фурье F_r и F_theta и решаем нерезонансное гомологическое уравнение для каждого нерезонансного коэффициента. Объект отклика хранит полную деформацию и ранжирование вкладов; ранжирование является диагностикой, а используемая ниже деформация - это полный отклик nRMP.

def helical_ripple_delta_B(eq_case):
    """Return the analytic helical-ripple contribution used by simple_stellarator."""
    def delta_B_helical(R, Z, phi):
        R_arr = np.asarray(R, dtype=float)
        Z_arr = np.asarray(Z, dtype=float)
        phi_arr = np.asarray(phi, dtype=float)
        theta = np.arctan2(Z_arr, R_arr - eq_case.R0)
        psi = eq_case.psi_ax(R_arr, Z_arr)
        dBR = eq_case.epsilon_h * eq_case.B0 * psi * np.cos(eq_case.m_h * theta - eq_case.n_h * phi_arr)
        return np.array([
            dBR,
            np.zeros_like(dBR, dtype=float),
            np.zeros_like(dBR, dtype=float),
        ])
    return delta_B_helical

[ORDER_ANALYSIS] Проверки порядка возмущений#

Переиспользуемый рабочий процесс теперь находится в нескольких небольших утилитах: scan_nonresonant_residual_order, scan_rmp_amplitude_order, scan_rmp_phase_order, scan_rmp_resolution_convergence и plot_perturbation_order_summary.

Ожидаемые порядки просты после фиксации соглашения Фурье:

  • Остаток формы поверхности nRMP: деформация первого порядка должна оставлять остаток карты O(k^2).

  • Резонансный коэффициент RMP: для delta B = k f линейность Фурье дает |b_{m,-n}| ~ k.

  • Ширина острова: маятниковая ширина Резерфорда/Нардона масштабируется как w ~ sqrt(|b_{m,-n}|), значит w ~ k^{1/2}.

  • Фаза X/O: фаза управляется arg(b_{m,-n}), а не амплитудой. Точное соотношение: m*Delta theta_O + Delta arg(b_{m,-n}) = 0.

Для проверки порядка фазового управления используется бездивергентный шаблон radial_rmp_field_template. Его фазовый параметр меняет фазу резонансного коэффициента, сохраняя div(delta B)=0, включая важный случай m=1. Мы намеренно тестируем слабо нелинейную управляющую фазу alpha(k)=k+eta*k^2; поэтому остаток относительно первого порядка по исходному k должен масштабироваться как O(k^2).

def deformed_torus_map_residual(epsilon_h, n_alpha=12):
    eq_case = simple_stellarator(
        R0=eq.R0, r0=eq.r0, B0=eq.B0,
        q0=eq.q0, q1=eq.q1,
        m_h=eq.m_h, n_h=eq.n_h, epsilon_h=float(epsilon_h),
    )
    psi_res = eq_case.resonant_psi(base_m, base_n)[0]
    deformation, r_res, _ = helical_velocity_deformation(
        eq_case, psi_res, n_theta=128, n_phi=128, include_shear=True
    )
    iota = 1.0 / float(eq_case.q_of_psi(psi_res))

    def surface(alpha, phi):
        alpha_arr = np.asarray(alpha)
        return (
            r_res + deformation.section_r(alpha_arr, phi),
            alpha_arr + deformation.section_theta(alpha_arr, phi),
        )

    def rhs(phi, state):
        radius, theta = state
        R = eq_case.R0 + radius*np.cos(theta)
        psi_here = (radius / eq_case.r0)**2
        q_here = float(eq_case.q_of_psi(psi_here))
        Bphi = eq_case.B0 * eq_case.R0 / R
        delta_BR = eq_case.epsilon_h * eq_case.B0 * psi_here * np.cos(eq_case.m_h * theta - eq_case.n_h * phi)
        return [
            R * delta_BR * np.cos(theta) / Bphi,
            1.0/q_here - R * delta_BR * np.sin(theta) / (radius * Bphi),
        ]

    остаток = deformed_surface_map_residual(
        surface,
        rhs,
        iota,
        alpha_values=np.linspace(0.0, 2*np.pi, n_alpha, endpoint=False),
        state_to_cartesian=lambda state, phi: [
            eq_case.R0 + float(state[0])*np.cos(float(state[1])),
            float(state[0])*np.sin(float(state[1])),
        ],
    )
    return остаток.max_residual

[ISLAND_WIDTHS] Столбчатая диаграмма ширин островов и схема перекрытия Чирикова#

Параметр перекрытия Чирикова определяется как

\[\sigma = \frac{w_1 + w_2}{|r_1 - r_2|}\]

где \(w_i\) - полуширины, а \(r_i\) - радиальные положения соседних островов. Стохастический транспорт начинается при \(\sigma \gtrsim 1\).

[MAGNETIC_SPECTRUM_ATLAS] Атлас многокомпонентного спектра контравариантного \(B^r\)#

Одиночная тепловая карта поверхности выше полезна для проверки доминирующей строки RMP, но производственный анализ магнитной топологии обычно требует семейства поверхностей. Здесь строится бездивергентное многокомпонентное возмущение, вычисляется спектр контравариантного радиального поля в стиле Нардона \(\tilde b^1_{mn}=\delta B^1/B_0^3\) на радиальном стеке и совместно анализируются все запрошенные резонансные строки.

Утилиты ниже намеренно модульны. Показанный атлас использует более широкое знаковое окно Фурье, 97 радиальных поверхностей, amplitude_scale='sqrt' и белые маски для нулевых/отсутствующих строк. Нелогарифмическая цветовая карта начинается с той же белой базы, поэтому почти нулевые коэффициенты не выглядят искусственно отделенными от маскированного фона. Знаковые оси показывают фактические строки Фурье в вычисленном спектре: (m,n) сопряжена с (-m,-n) для вещественного поля, но (m,n) и (m,-n) являются независимыми строками противоположной винтовости, если модель возмущения не накладывает дополнительную симметрию. amplitude_scale='log10' остается полезной, когда нужна строгая проверка динамического диапазона, а не визуальное распознавание паттернов.

  • plot_rational_surface_map объединяет необязательный q-профиль, маркеры рациональных поверхностей низкого порядка, проецированные точки Пуанкаре и полосы ширины островов в плоскости (m/n, s).

  • plot_spectrum_heatmap(..., renderer='pcolormesh') - рекомендуемый вид спектра по поверхностям; он может наложить физическую резонансную ветвь m=-q n_F без отрисовки ветви противоположной винтовости.

  • plot_spectrum_bar3d возвращает интерактивную фигуру Plotly для вращения, масштабирования и инспекции доминирующих строк при наведении.

  • plot_radial_mode_heatmap(fixed_n=..., resonant_sign=+1) отслеживает все строки \(m\) при фиксированном Фурье-\(n\) и рисует ветвь положительного q в полуплоскости отрицательных \(m\), \(m=-nq(s)\).

  • plot_radial_mode_heatmap(fixed_m=..., axis_convention='fourier') отслеживает фактические строки Фурье \(n\) и рисует ветвь положительного q \(n=-m/q(s)\); q-кривая и полосы ширины островов остаются независимо включаемыми.

Желтые вертикальные полосы отмечают оценку полуширины Нардона по одному коэффициенту на рациональной поверхности низкого порядка; их толщина масштабируется с амплитудой резонансного коэффициента. Это необязательное наложение, а не часть самой тепловой карты. На каждой радиальной карте рисуется только физически резонансная кривая. Ветвь противоположной винтовости не является сопряженной диагностикой и намеренно не зеркалируется здесь. Если несколько гармоник имеют одну и ту же сокращенную рациональную поверхность, истинную ширину острова при конечной амплитуде следует вычислять из объединенного резонансного гамильтониана; полосы по отдельным строкам являются диагностикой, а не этой нелинейной суммарной шириной.
