Draw Poincare points colored by a scalar attached to each seed orbit.

background must expose section_points(section_index) returning (R, Z, seed_index). Finite seed values are colored by cmap; seeds with infinite values can be rendered in nonfinite_color so long-lived and wall-hitting populations remain visually distinct without retracing.
