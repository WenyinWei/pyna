pyna.dynamics#

General-purpose dynamical systems for pyna.

This module complements the magnetic-confinement focused pyna.topo stack with small, reusable building blocks for broad dynamical-systems work: callable flows, Hamiltonian systems, pairwise N-body systems, arbitrary finite-dimensional maps, and Ito SDEs.
