pyna.toroidal.pest_coords#

Utilities for building toroidal flux-surface coordinate meshes.

This module contains data-format-independent pieces used when converting ordered Poincare traces into straight-field-line theta grids. Field-line tracing remains in pyna.toroidal.flt/cyna; these helpers operate on already collected crossings and axis samples.

Interpolate scattered periodic samples onto a target theta grid.

Duplicate theta bins, after rounding, are averaged before interpolation. The interpolation is periodic by appending one wrapped sample at each end.

pyna.toroidal.pest_coords.circle_map_lift_iota(phase: numpy.ndarray, *, max_iota: float = 0.5, min_points: int = 16, grid_size: int = 1001, chunk: int = 128, xatol: float = 1e-10) → tuple[float, float][source]#

Fit phase_k = phase0 + 2*pi*iota*k directly on the circle.

The input phases remain modulo 2*pi; the turn index k supplies the covering-space information. Both phase orientations are tried and the concentration with the larger circular score is retained.

Returns:

(iota, circular_rms) – nan values are returned when the phase sequence is insufficient.
