Recompute section-local DP^m for existing fixed points.

Orbit tracing can clone one source DPm onto every map-ordered point. That preserves eigenvalues but not local eigenvectors. This helper runs a short Newton refinement at each supplied point using the same field-period span, so manifold seeds can use the correct local stable/unstable directions at every P^k point.

pyna.toroidal.flt.island_chain.deduplicate_boundary_island_orbits(orbits: Sequence[BoundaryIslandOrbit], *, orbit_dedup_tol: float = 0.0005, start_orbit_id: int = 0, chain_id: int | None = None, winding: tuple[int, int] | None = None, reduced_winding: tuple[int, int] | None = None) → tuple[BoundaryIslandOrbit, Ellipsis][source]#

Deduplicate equivalent fixed-point orbits and assign stable IDs.

The equivalence test is phase-invariant along the map orbit: orbits with the same ordered point set but different starting fixed points are treated as one physical orbit. Returned orbits are annotated with orbit_id and their points carry matching orbit_id, point_index and same_orbit_key metadata so section cuts can be identified across plots.

Trace ordered orbits from converged fixed points using P_span.

This is a batch wrapper around the existing cyna arbitrary-span map tracer. For each source point it stores the m distinct orbit points and summarizes the closing endpoint with closure_residual. Set deduplicate=True to collapse equivalent orbits from different seeds and attach stable same-orbit metadata for cross-section plotting.

Trace W^u/W^s point clouds from hyperbolic fixed points.

W^u is traced forward from the unstable eigendirection. W^s is traced with the backward numerical map; by default its local anchor is refined against that inverse map before seeds are generated. This avoids silently using a forward-map fixed point that is not also a fixed point of the discrete backward RK map at finite step size. Pass map_span when the supplied DPm is already the monodromy of a complete return map. If map_span is omitted, the traced span is inferred from fixed-point metadata as map_power * field_period.
