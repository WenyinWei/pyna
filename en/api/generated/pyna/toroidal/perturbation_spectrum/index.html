Return O/X poloidal angles for one or more toroidal sections.

The convention is the Nardon expansion tilde_b^1 = sum b_mn exp(i(m theta* + n phi)). For the resonant coefficient b_{m,-n}, fixed points satisfy m theta* - n phi + arg(b_{m,-n}) = +/- pi/2.

Compute B^1 = B dot grad(s) and B^3 = B dot grad(phi).

R_surf and Z_surf must be a radial stack with shape (n_phi, n_r, n_theta). The returned arrays have the same stripped shape, after any duplicated periodic endpoints have been removed. Cylindrical field components use the physical orthonormal basis (e_R, e_phi, e_Z).

Compute Nardon’s tilde_b^1 = delta B^1 / B_0^3 on surfaces.

Pass denominator_B_phi when the denominator should be the background toroidal contravariant field B_0 dot grad(phi). Pass denominator_B3 directly if it is already available on the same (phi, radial, theta) surface grid.

pyna.toroidal.perturbation_spectrum.radial_perturbation_Fourier_spectrum(dBr_grid: numpy.ndarray, theta_vals: numpy.ndarray, phi_vals: numpy.ndarray, *, radial_labels: numpy.ndarray | None = None, m_max: int | None = None, n_max: int | None = None, min_amplitude: float = 0.0) → RadialPerturbationFourierSpectrum[source]#

Compute dBr_mn for f(theta, phi)=sum dBr_mn exp(i(m theta+n phi)).

dBr_grid may be a single surface with shape (n_phi, n_theta), a radial-first stack (n_r, n_phi, n_theta), or the phi-first stack (n_phi, n_r, n_theta) returned by radial_perturbation_component(). For radial stacks, dBr has shape (n_r, n_modes).

pyna.toroidal.perturbation_spectrum.island_chain_fixed_points(m: int, n: int, coefficient: complex, phi: float | numpy.ndarray, *, q_prime_sign: int = 1) → dict[str, numpy.ndarray][source]#

Return O/X poloidal angles implied by tilde_b^1_{m,-n}.

The returned theta_O and theta_X arrays have shape (n_phi, m). A phase change coefficient *= exp(1j * alpha) rotates every branch by -alpha / m at fixed toroidal section.

pyna.toroidal.perturbation_spectrum.nardon_island_half_width(q: float, q_prime: float, m: int, b_res: float) → float[source]#

Return Nardon’s magnetic-island half-width in the radial coordinate.

The thesis formula is sqrt(4 q^2 b_res / (q' m)). This implementation returns a positive geometric width and therefore uses abs(q' m) in the denominator.

pyna.toroidal.perturbation_spectrum.analyze_resonant_island_chains(spectrum: RadialPerturbationFourierSpectrum, q_profile: numpy.ndarray, *, n: int, radial_labels: numpy.ndarray | None = None, m_values: Iterable[int] | None = None, min_b_res: float = 0.0) → list[ResonantIslandChain][source]#

Analyze resonant island chains from a radial Fourier spectrum.

For each requested m this finds roots of q(s) = m/n, interpolates the resonant coefficient tilde_b^1_{m,-n}, and evaluates Nardon’s island half-width formula in the same radial coordinate s.

pyna.toroidal.perturbation_spectrum.analyze_resonant_island_chains_multi_n(spectrum: RadialPerturbationFourierSpectrum, q_profile: numpy.ndarray, *, n_values: Iterable[int] | None = None, radial_labels: numpy.ndarray | None = None, m_values: Iterable[int] | collections.abc.Mapping[int, Iterable[int]] | None = None, min_b_res: float = 0.0) → list[ResonantIslandChain][source]#

Analyze all requested resonant (m, n) island chains together.

This is the multi-component counterpart to analyze_resonant_island_chains(). For each positive toroidal mode number n it finds every requested q(s)=m/n crossing, interpolates the resonant coefficient tilde_b^1_{m,-n}, and returns one combined list sorted by radial position and mode number.

Parameters:

  • spectrum – Radial stack of tilde_b^1_{mn} Fourier coefficients.

  • q_profile – Safety-factor profile sampled on radial_labels.

  • n_values – Positive physical toroidal mode numbers to scan. If omitted, all positive abs(n) values present in spectrum are scanned.

  • radial_labels – Optional radial labels overriding spectrum.radial_labels.

  • m_values – Optional positive poloidal mode numbers. Pass a mapping {n: m_list} when different toroidal families need different candidate m values.

  • min_b_res – Drop chains with 2*abs(tilde_b^1_{m,-n}) below this threshold.
